The GreenCharge app that makes charging your battery powered devices greener.

GreenCharge uses a smart plug to help charging in a green way. This implementation uses the Shelly Plug S. First you need to register the device you want to charge and provide details about the smart plug. After registering a one time calibration charge run needs to be done, so the system knows the device battery's characteristics. From then on you can charge your device with GreenCharge. Lastly you can see the avoided emissions by using GreenCharge
